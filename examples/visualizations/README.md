Visualization runner

This folder contains the HTML output generated by the chunking visualizer and instructions to reproduce it.

Quick steps (local machine):

1. Generate the visualization HTML from a text file:

```bash
# uses ts-node loader as configured in package.json
npm run visualize --silent -- path/to/your.txt
# or
node --loader ts-node/esm scripts/visualize_chunking.ts path/to/your.txt
```

By default the script writes to `examples/visualizations/chunk_view.html`.

2. (Optional) Render the HTML to PNG using Playwright:

```bash
# install Playwright browsers (required once)
npx playwright install

# render the generated HTML to PNG
npm run visualize:render --silent -- examples/visualizations/chunk_view.html examples/visualizations/chunk_view.png
# or
node --loader ts-node/esm scripts/render_visualization.ts examples/visualizations/chunk_view.html examples/visualizations/chunk_view.png
```

Notes:
- The project uses `ts-node` to run TypeScript scripts directly. If your environment doesn't support the `--loader ts-node/esm` invocation, use `npx ts-node` or compile to JavaScript with `tsc`.
- Rendering with Playwright requires the browser binaries (use `npx playwright install`).
- If you run into Node loader errors, ensure `ts-node` is installed (`npm ci`) and that your Node.js version is compatible.
